# AI Assistant Comparison

A comparison project between:

- Open Source AI Assistant (Qwen2.5-0.5B)
- Frontier AI Assistant (Gemini API)

## Features

- Multi-turn conversations
- Separate assistant memory
- Safety guardrails
- Latency tracking
- Analytics dashboard
- Interaction logging

## Tech Stack

- Streamlit
- Hugging Face Transformers
- Google Gemini API
- Python

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
